## üîÆ Virtual Environment

<Fragment>
  A virtual environment is established to manage <PythonIcon /> Python
  dependencies.
</Fragment>

It needs creation only once but can be activated multiple times.

:::tip
You will need to repeat this step each time you open a new terminal session.
:::

<Fragment>
  ### `venv` on <WindowsIcon /> Windows

#### Create `venv`

Execute the following command to create a virtual environment:

```bash
python3.10 -m venv venv
```

#### Activate `venv`

Run this command to activate the virtual environment:

```bash
. venv/bin/activate
```

  ## üß© Modules <SizeInfo size={'~14 GB'} />

<Fragment>
  Modules constitute the components of <AppIcon /> {APP_NAME} and occupy
  approximately 14 GB of disk space.

<Fragment>
  ### Modules on <WindowsIcon /> Windows
</Fragment>

Install all modules sequentially using the following command:

```powershell
pnpm i; pnpm install:processing:windows
```

:::tip
Use right-click to paste in Windows terminals.
:::

<Fragment>

:::tip
For accelerated processing with
a <NvidiaIcon /> [Nvidia](https://www.nvidia.com/) compatible GPU, include
additional
dependencies:

```bash
pnpm install:processing:cuda:windows
```

:::

</Fragment>

<Fragment>
  ### Modules on <UbuntuIcon /> Ubuntu / <MacOsIcon /> macOS
</Fragment>

Install all modules at once with this command:

```bash
pnpm i && pnpm install:processing
```

<Fragment>

:::tip
For accelerated processing with
a <NvidiaIcon /> [Nvidia](https://www.nvidia.com/) compatible GPU, include
additional
dependencies:

```bash
pnpm install:processing:cuda
```

:::

</Fragment>
