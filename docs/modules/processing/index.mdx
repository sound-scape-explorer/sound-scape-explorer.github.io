# 🧮 Processing

Written in <PythonIcon /> Python 3, this module harnesses machine learning and
a range of algorithms to extract valuable metrics from audio recordings.

The generated data is stored in <StorageIcon /> `.h5` format, allowing seamless
integration with the Visualisation module and enhancing data sharing among
users.
