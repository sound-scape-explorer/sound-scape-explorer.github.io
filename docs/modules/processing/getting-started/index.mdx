# 🏃 Getting started

## 📋 Requirements

You are able to generate data if you have:

- Ran the 🔧 [Installation](../../../installation)
- Have a complete <ExcelIcon /> configuration
- Have a terminal window with active <PythonIcon /> Python `venv`

:::note
[Learn how to activate 🔮 virtual
environments](../../../installation/application#-virtual-environment)
:::

## 🚀 Run

1. Type <AppInline /> followed by a `<space>`
2. Type the path to your `.xlsx` file (or drag and drop file icon to the
   terminal)
3. Press `Enter`
