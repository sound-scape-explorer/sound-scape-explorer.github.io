## Most frequent use case

If you want to execute all the requested metrics declared in your configuration
file, please choose `Run All` in the menu.

1. Select `Run All`
2. Wait for the processing to complete. When the menu displays again, the
   processing has completed
3. If you intend to visualise your data, choose `Quit` then proceed to the
   [visualisation user guide](../../visualisation)
