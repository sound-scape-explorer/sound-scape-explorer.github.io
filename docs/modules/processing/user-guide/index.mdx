## 📋 Most Common Use Case

<Fragment>
  For executing all metrics specified in your <ExcelIcon /> configuration file:
</Fragment>

1. Choose `Run All`
2. Wait for processing completion. When the menu reappears, the
   processing has finished
3. To visualise your data, select `Quit`, then proceed to the 👁️‍🗨️ [Visualisation User Guide](../../visualisation/)
