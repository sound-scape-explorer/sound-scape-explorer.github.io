# 🏃 Getting started

:::tip
Paths described in the <SimpleIcon icon="microsoftexcel" color="#217346" />
configuration are **relative** to this file.

We recommend creating **one folder per campaign** that will store your
configuration and storage files.
:::

## 📥 Download

Download the default <SimpleIcon icon="microsoftexcel" color="#217346" /> <ExcelFile version={VERSION}>v{VERSION} configuration</ExcelFile>

## 🏠 Local

1. Navigate to <ExampleDirectory version={VERSION} />
2. Open the `new-campaign` folder
3. Copy and edit the default <SimpleIcon icon="microsoftexcel" color="#217346" /> `config.xlsx`
