## <ExcelIcon/> _Excel_ configuration

:::tip
Paths referenced in the <ExcelIcon /> configuration are **relative** to this
file.
:::

The <ExcelIcon /> configuration is divided in multiple tabs:

### <ExcelTab background="#eeeeee">Help</ExcelTab>

This tab provides quick information.

### <ExcelTab background="#ffffa6">Settings</ExcelTab>

Specify your campaign settings in this tab.

#### `audio_path` Important Note

:::danger
For <WindowsIcon /> Windows users, ensure that the `audio_path` does **not**
contain spaces to avoid disrupting the audio server's functionality.

If you've already generated processing data, rectify the `audio_path` within
your configuration file and **refresh the configuration** to implement the
changes.

This issue is known and a fix is currently under development.

Apologies for any inconvenience caused!
:::

### <ExcelTab background="#ffffa6">Files</ExcelTab>

Specify file paths, sites, dates, and labels in this tab.

### <ExcelTab background="#ffffa6">Bands</ExcelTab>

Specify your filtering bands in this tab.

### <ExcelTab background="#ffffa6">Integrations</ExcelTab>

Specify integration intervals in this tab.

### <ExcelTab background="#ffffa6">Ranges</ExcelTab>

Specify dates and time spans in this tab.

### <ExcelTab background="#ffffa6">Trajectories</ExcelTab>

Specify trajectories in this tab.

### <ExcelTab background="#b4c7dc">Extractors</ExcelTab>

Configure extraction settings in this tab.

Refer to the <ExcelTab background="#eeeeee">ListExtractors</ExcelTab> tab for
available extractors.

### <ExcelTab background="#b4c7dc">Digesters</ExcelTab>

Configure _digesters_ (metrics) in this tab.

Refer to the <ExcelTab background="#eeeeee">ListDigesters</ExcelTab> tab for
available digesters.

### <ExcelTab background="#b4c7dc">Reducers</ExcelTab>

Configure reduction settings in this tab.

Refer to the <ExcelTab background="#eeeeee">ListReducers</ExcelTab> tab for
available reducers.

### <ExcelTab background="#b4c7dc">Autoclusters</ExcelTab>

Configure autoclustering settings in this tab.

Refer to the <ExcelTab background="#eeeeee">ListAutoclusters</ExcelTab> tab for
available autoclusters.
