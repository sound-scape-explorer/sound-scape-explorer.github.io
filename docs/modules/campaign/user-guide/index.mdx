#### `audio_path` important note

:::danger
The `audio_path` on Windows **must not** contain spaces at the risk of breaking
the audio server.

If you already generated your processing data, you can fix the `audio_path`
inside your configuration file and refresh the configuration to see the
changes.

This is a known blocking issue and a fix is being worked on.

Sorry for the inconvenience!
:::
