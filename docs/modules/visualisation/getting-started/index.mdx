# 🏃 Getting started

<Fragment>
  :::warning
  Ensure you have an active <PythonIcon /> Python `venv` in a terminal window.
  :::

</Fragment>

:::info
[Learn how to activate 🔮 virtual
environments](../../../installation/application#-virtual-environment)
:::

The 👁️‍🗨️ Visualisation module uses two services to run properly:

- `Front` service
- Power the user interfaces
- The default port is `5530`
- `Audio` service
- Serve audio files from your file system
- The default port is `5531`

## 🔗 Compatibility

<Fragment>
  While the web app is compatible with most browsers, we've encountered
  intermittent issues with <FirefoxIcon /> Firefox on <WindowsIcon /> Windows 11
  and <SafariIcon /> Safari on <MacOsIcon /> macOS 14. For optimal performance,
  we recommend using <ChromeIcon /> Chrome.
</Fragment>

## 🔧 `sse_audio` Command

Use the `sse_audio` command to start only the `Audio` service.

This can be useful in case you are using the 👁️‍🗨️ [Visualisation online](../../../extras/visualisation-online/).

<AudioCommandBlock />

## 🔧 Advanced Commands

### `audio:front` Command

Simultaneously start both the `Front` and `Audio` services.

```bash
pnpm audio:front relative/or/absolute/path/to/audio/folder
```

### `audio` Command

Start the `Audio` service on port [5531](http://localhost:5531).

```bash
pnpm audio -- relative/or/absolute/path/to/audio/folder
```

### `front` Command

Start the `Front` service on port [5530](http://localhost:5530).

```bash
pnpm front
```
