# Campaign guide

Creating a campaign is the first user step in using {APP_DESCRIPTION}.

A campaign will allow you to add audio files, describe user labels and define
intervals and metrics.

## Procedure

1. Go to <ExampleDirectory version={VERSION} />
2. Duplicate the `new-campaign` folder, move it and rename it
3. Open your new folder
4. Edit the `.xlsx` configuration file
5. You can then use the [processing user
   guide](./processing) to generate your data
