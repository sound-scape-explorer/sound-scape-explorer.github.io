# Processing user guide

:::warning

Your must have an **active** python `venv`. Find how to [activate virtual
environment](../venv)

:::

## Run

1. Type <AppInline /> followed by a `<space>`
2. Type the path to your `.xlsx` file (or drag and drop file icon to the
   terminal)
3. Press `Enter`

<ProcessingCommandBlock />

## Menu

You will get the menu below.

Use your `arrow` and `Enter` keys to navigate through it.

<ProcessingMenuBlock />

## Most frequent use case

If you want to execute all the requested metrics declared in your configuration
file, please choose `Run All` in the menu.

1. Select `Run All`
2. Wait for the processing to complete. When the menu displays again, the
   processing has completed
3. If you intend to visualise your data, choose `Quit` then proceed to the
   [visualisation user guide](../visualisation.mdx)
