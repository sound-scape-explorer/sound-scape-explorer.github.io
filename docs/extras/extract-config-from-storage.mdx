# 📦 Extract <ExcelIcon /> Config from <StorageIcon /> Storage

:::warning
Ensure you have an active <PythonIcon /> Python `venv` in a terminal window.
:::

:::info
[Learn how to activate 🔮 virtual
environments](../installation/application#-virtual-environment)
:::

## 🔍 Context

This command allows the extraction of the <ExcelIcon /> configuration file from
an existing <StorageIcon /> storage file.

The resulting `storage.config-{VERSION}.xlsx` file represents the <ExcelIcon />
configuration used in creating the <StorageIcon /> storage file.

## 🚀 Execution

1. Type <ConfigCommandInline/>, then add a `<space>`
2. Drag and drop your <StorageIcon /> storage file into the terminal (or type
   path)
3. Press `Enter`
