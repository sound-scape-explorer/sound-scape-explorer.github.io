🧭 Menu

Import​

Load a local .h5 storage file.

Settings​

Change the user interface settings.

Help​

Find help links and keyboard shortcuts.

Selection​

Select your band, integration, extractor and reducer.

Colors​

Color intervals based on index, timestamp, label.

Labels​

Filter intervals based on their labels.

Time​

Filter intervals based on their timestamps.

Details​

Show details about the selected interval.

Audio​

Show audio for the selected interval.

Trajectories​

Display trajectories.

Indicators​

Display acoustic indicators.

Digested​

Display digested metrics.
