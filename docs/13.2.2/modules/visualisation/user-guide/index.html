📖 User Guide

The visualisation interface relies on floating windows to organize the available viewport efficiently.

In a minimalist design, each window can be effortlessly toggled between hidden or shown states by interacting with the menu icons.

Discover more about the 🧭 Menu and 📊 Plot controls for additional help.

🧭 Menu​

IconFunctionDescriptionKeyboard Shortcut
ImportLoad a local .h5 storage file.i
SettingsAdjust user interface settings.,
HelpAccess help links and keyboard shortcuts.h
SelectionSelect band, integration, extractor, and reducer.s
ColorsColour intervals by index, timestamp, or label.c
LabelsFilter intervals by labels.z
TimeFilter intervals by timestamps.t
DetailsShow details about the selected interval.d
AudioShow audio for the selected interval.a
TrajectoriesDisplay trajectories.x
Relative TrajectoriesDisplay relative trajectories.r
IndicatorsDisplay acoustic indicators.v
DigestedDisplay digested metrics.b

📊 Plot controls​

IconFunctionDescription
ZoomSet cursor to zoom mode.
PanSet cursor to pan mode.
OrbitalSet cursor to orbital rotation mode.
TurntableSet cursor to turntable rotation mode.
Reset cameraReset camera to default position.
Download PNGDownload scatter as PNG.
Download SVGDownload scatter as SVG.
Eport CSVExport scatter as CSV.

⏳ Loading a new configuration​

tip

The 👁️‍🗨️ Visualisation module can handle only one configuration file at a time.

You will need to stop current services in order to load a new one.
