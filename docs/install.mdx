# Installing {APP_NAME}

:::tip

You can install all modules at once by using these variants:

- `pnpm install:all`
- `pnpm install:all:windows`
- `pnpm install:all:cuda`
- `pnpm install:all:cuda:windows`

:::

## Windows

1. Download <SourceLink version={VERSION} size={SIZE} />
2. Extract archive (use _Extract here_)
3. Open folder <AppDirectory version={VERSION} />
4. Double click on `start.bat` (wait for `venv` creation)
5. Enter command `pnpm i`
6. Enter command `pnpm install:front`
7. Enter command `pnpm install:processing:windows`

   :::tip
   Have CUDA enabled graphics?

   Enter command `pnpm install:processing:cuda:window`
   :::

8. Close the shell
9. You can now [run a campaign](./user-guides/running-campaigns.mdx)

## Ubuntu / macOS

1. Download <SourceLink version={VERSION} size={SIZE} />
2. Extract archive (use _Extract here_)
3. Open folder <AppDirectory version={VERSION} />
4. Open a new terminal
5. Type `cd` followed by a `<space>` and the path to the <AppDirectory
   version={VERSION} /> folder

   :::tip
   Drag and drop the folder icon into your terminal to avoid typing the path!
   :::

6. Enter command `python3.10 -m venv venv`
7. Enter command `. venv/bin/activate`
8. Enter command `pnpm i` <SizeInfo size={'~50 MB'} />
9. Enter command `pnpm install:front` <SizeInfo size={'~500 MB'} />
10. Enter command `pnpm install:processing` <SizeInfo size={'~4 GB'} />

    :::tip
    Have CUDA enabled graphics?

    Enter command `pnpm install:processing:cuda` <SizeInfo size={'~2 GB'} />
    :::

11. Close the terminal
12. You can now [run a campaign](./user-guides/running-campaigns.mdx)
