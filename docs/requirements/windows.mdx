# Installing on Windows

Compatible with Windows 10, and Windows 11.

## Installation procedure

1. Click on the start menu (or press <WinKey/> button on your keyboard)
2. Type `PowerShell`
3. Right-click on the `PowerShell` icon
4. Select `Run as Administrator` (this will open a new shell window)
5. Navigate to [chocolatey installation page](https://chocolatey.org/install)
6. Scroll to the installation command
7. Copy the command
8. Paste it in your shell window
9. Press `Enter` (Wait and type `choco --version` to verify installation)
10. Install `Python`:

```powershell
choco install python3 --version=3.10.11 -y
```

> **Note**
>
> If you have a prior version of Python and want to uninstall it. Run the
> following command
>
> ```powershell
> choco uninstall python3
> ```

11. Install `Node.js`:

```powershell
choco install nodejs --version=16.20.0 -y
```

12. Install `ffmpeg`, `curl` and `pnpm`:

```powershell
choco install ffmpeg curl pnpm -y
```

13. Close the shell window.
14. Navigate to [Microsft C++ Build
    Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
15. Click on `Download Build Tools` to download the executable
16. Run the executable (continue until you get the menu _Installing - Visual
    Studio Build Tools..._)
17. Tick `Desktop development with C++`
18. Press `Install` (6 Gb of disk space will be used)
19. Wait for completion, you can close the Visual Studio Build Tools window
20. You can now [install {APP_NAME}](../install)
