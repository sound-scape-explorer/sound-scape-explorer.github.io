# <MacOsIcon /> macOS

The steps outlined below have been successfully tested on macOS 13 and macOS 14.

## <HomebrewIcon /> Homebrew

Using the package manager <HomebrewIcon /> [Homebrew](https://brew.sh/) is
recommended to install the application requirements

1. Navigate to [brew.sh](https://brew.sh/)
2. Follow the installation instructions provided for <HomebrewIcon /> Homebrew
3. Open a <TerminalIcon /> terminal to proceed with the next steps

## üì¶ Dependencies

Execute the following command to install dependencies and update your path:

- <CIcon /> [C compiler](https://gcc.gnu.org/)
- <CurlIcon /> [curl](https://curl.se/)
- <FfmpegIcon /> [ffmpeg](https://www.ffmpeg.org/)
- <PythonIcon /> [Python](https://www.python.org/)
- <NodeIcon /> [Node.js](https://nodejs.org/)
- <PnpmIcon /> [pnpm](https://pnpm.io/)

```bash
brew install gcc curl ffmpeg python@3.10 node@20 pnpm && \
echo 'export PATH="/usr/local/opt/node@20/bin:$PATH" >> ~/.profile'
```
