# <WindowsIcon /> Windows

The following steps have been tested on Windows 10, and Windows 11

## <CIcon /> C compiler

In order to install the
[hdbscan](https://github.com/scikit-learn-contrib/hdbscan) python package later
on, a C compiler will be required

You can install it on Windows through the [Visual Studio Build
Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)

### üõ†Ô∏è Procedure

1. Navigate to <MicrosoftIcon /> [Microsft C++
   Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
2. Click on `Download Build Tools` to download the executable
3. Run the executable (continue until you get the menu _Installing - Visual
   Studio Build Tools..._)
4. Tick `Desktop development with C++`
5. Press `Install` (6 Gb of disk space will be used)
6. Wait for completion, you can close the Visual Studio Build Tools window

## <ChocolateyIcon /> Chocolatey

Using the package manager <ChocolateyIcon />
[Chocolatey](https://chocolatey.org/) is recommended to install the application
requirements

### üõ†Ô∏è Procedure

1. Click on the start menu (or press <WinKey /> button on your keyboard)
2. Type _<PowershellIcon /> PowerShell_
3. Right-click on the _<PowershellIcon /> PowerShell_ icon
4. Select _Run as Administrator_ (this will open a new elevated terminal)
5. Navigate to <ChocolateyIcon /> [Chocolatey installation page](https://chocolatey.org/install)
6. Scroll to the installation command
7. Copy the command
8. Paste it in your terminal
9. Press `Enter` (wait for the installation to complete)
10. Keep the terminal open and process through the next steps

## <PythonIcon /> Python

```powershell
choco install python3 --version=3.10.11 -y
```
