## üîÆ Virtual environment

A virtual environment is created where <PythonIcon /> Python dependencies will
be installed.

### `venv` on <WindowsIcon /> Windows

Double click on `start.bat` and wait for `venv` creation and activation.

### `venv` on <UbuntuIcon /> Ubuntu / <MacOsIcon /> macOS

1. Open a new terminal
2. Type `cd` followed by a `<space>` and the path to the <AppDirectory
       version={VERSION} /> folder
   :::tip
   You can drag and drop the folder icon into your terminal instead of typing.
   :::

#### Create `venv`

Enter the following command to create a virtual environment

```bash
python3.10 -m venv venv
```

#### Activate `venv`

Enter the following command to activate the virtual environment

```bash
. venv/bin/activate
```

## üß© Modules <SizeInfo size={'~14 GB'} />

### Modules on <WindowsIcon /> Windows

Install all modules at once by using the following command

```powershell
pnpm i; pnpm install:front; pnpm install:processing:windows
```

:::tip
If you have a compatible GPU, you can install extra dependencies to accelerate
processing

```bash
pnpm install:processing:cuda:windows
```

:::

### Modules on <UbuntuIcon /> Ubuntu / <MacOsIcon /> macOS

Install all modules at once by using the following command

```bash
pnpm i && \
pnpm install:front && \
pnpm install:processing
```

:::tip
If you have a compatible GPU, you can install extra dependencies to accelerate
processing

```bash
pnpm install:processing:cuda
```

:::
