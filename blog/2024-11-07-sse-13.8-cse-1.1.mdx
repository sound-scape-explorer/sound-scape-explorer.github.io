# New SSE 13.8.0 and CSE 1.1.0

<Fragment>
  This dual release for <SseIcon/> SSE version 13.8.0 and <CseIcon/> CSE
  version 1.1.0 adds new nn-based extractors (`melogram` and `melspectrum`).

**Melogram** is now used as the default nn extractor, thus replacing **VGGish**.

</Fragment>
