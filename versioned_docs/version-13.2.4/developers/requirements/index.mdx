### 🔧 Installation

<Fragment>
  Use the <ChocolateyIcon /> Chocolatey command below in an elevated (admin){' '}

```powershell
choco install nodejs-lts pnpm -y
```

</Fragment>

```powershell
choco uninstall nodejs-lts pnpm -y
```

<Fragment>
  ## <UbuntuIcon /> Ubuntu
</Fragment>

### 🔧 Installation

<Fragment>
  Open a <TerminalIcon /> terminal window and run the commands below.
</Fragment>

<Fragment>
  Install <NodeIcon /> Node.js with the following command:
</Fragment>

```bash
sudo apt update && \
sudo apt install -y ca-certificates curl gnupg && \
sudo mkdir -p /etc/apt/keyrings && \
curl -fsSL https://deb.nodesource.com/gpgkey/nodesource-repo.gpg.key | sudo gpg --dearmor -o /etc/apt/keyrings/nodesource.gpg && \
echo "deb [signed-by=/etc/apt/keyrings/nodesource.gpg] https://deb.nodesource.com/node_20.x nodistro main" | sudo tee /etc/apt/sources.list.d/nodesource.list && \
sudo apt update && \
sudo apt install -y nodejs
```

<Fragment>
  Install <PnpmIcon /> pnpm with the following command:
</Fragment>

```bash
sudo npm -g i pnpm
```

### 🗑️ Uninstallation

<Fragment>
  Open a <TerminalIcon /> terminal window and execute the following command:
</Fragment>

```bash
sudo npm -g uninstall pnpm && \
sudo apt remove -y nodejs && \
sudo rm /etc/apt/sources.list.d/nodesource.list && \
sudo rm /etc/apt/keyrings/nodesource.gpg && \
sudo apt update && \
sudo apt autoremove -y
```

<Fragment>
  ## <MacOsIcon /> macOS
</Fragment>

### 🔧 Installation

<Fragment>
  Execute the following command in a <TerminalIcon /> terminal window to install
  and update your path:
</Fragment>

```bash
brew install node@20 pnpm && \
echo 'export PATH="/usr/local/opt/node@20/bin:$PATH" >> ~/.profile'
```

### 🗑️ Uninstallation

Open a <TerminalIcon /> terminal window
and execute the following command:

```bash
brew uninstall node@20 pnpm
```
