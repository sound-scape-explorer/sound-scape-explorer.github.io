import {ColorFill} from '@blueprintjs/icons';
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

Open the <Button icon={<ColorFill />} /> `Colors` panel and
access to the following controls:

- Color Category
  (<span style={{color: 'red'}}>1</span>)
  and Color Option
  (<span style={{color: 'red'}}>2</span>)
  - `DEFAULT`: color by the hour in a day, by day or night,
    or by interval index
  - `TAGS`: use tag values for coloring
  - `ACOUSTICS`: use acoustic values for coloring
- Opacity values
  (<span style={{color: 'red'}}>3</span>)
  - For **excluded** (hidden) points
  - For **included** (visible) points
- Color flavor (palette)
  (<span style={{color: 'red'}}>4</span>)
- Color map reverse button
  (<span style={{color: 'red'}}>5</span>)
  and gradients
  (<span style={{color: 'red'}}>6</span>)

:::tip

The gradient is rendered dynamically, relative to current
color category and option.

Hover the gradient color slices to display the actual underlying values.

:::
