# Installing {APP_NAME}

</Fragment>

:::tip

You can install all modules at once by using these variants:

- `pnpm install:all`
- `pnpm install:all:windows`
- `pnpm install:all:cpu`
- `pnpm install:all:cpu:windows`

:::

## Windows

<Fragment>
  <ol>
    <li>
      {'Open the '}
      <ReleaseLink version={VERSION} />
    </li>
    <li>Scroll to _Assets_</li>
    <li>Download the `Source code (zip)` file</li>
    <li>Extract the downloaded archive (Extract here)</li>
    <li>
      {'Open the extracted folder '}
      <AppDirectory />
    </li>
    <li>Double-click on `start.bat`. Wait for `venv` creation if needed</li>
    <li>Enter command `pnpm i`</li>
    <li>Enter command `pnpm install:front`</li>
    <li>
      {'Enter command '}
      <code>pnpm install:processing:windows</code>
      {'. If you get an '}
      <code>nvidia-*</code>
      {' error, please use command '}
      <code>pnpm install:processing:cpu:windows</code>
    </li>
    <li>Close the shell</li>
    <li>You can now [run a campaign](./user-guides/running-campaigns)</li>
  </ol>
</Fragment>
