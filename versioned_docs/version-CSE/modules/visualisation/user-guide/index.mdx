# üìñ User Guide

The visualisation interface relies on floating windows to organize the
available viewport efficiently.

In a minimalist design, each window can be effortlessly toggled between hidden
or shown states by interacting with the menu icons.

Discover more about the üß≠ [Menu](#-menu)
and üìä [Plot controls](#-plot-controls) for additional help.
