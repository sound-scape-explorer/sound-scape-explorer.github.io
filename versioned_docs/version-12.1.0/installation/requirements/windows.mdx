<Fragment>
  ## <CIcon /> C Compiler
</Fragment>

In order to install the
[hdbscan](https://github.com/scikit-learn-contrib/hdbscan) python package later
on, a C compiler will be required.

<Fragment>
  You can install it on <WindowsIcon /> Windows through the [Visual Studio Build
  Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/).
</Fragment>

### üõ†Ô∏è Procedure

<ol>
  <li>
    {'Navigate to '}
    <MicrosoftIcon />
    {'  '}
    <a
      href={'https://visualstudio.microsoft.com/visual-cpp-build-tools/'}
      target={'_blank' + ''}
    >
      {'Microsoft C++ Build Tools'}
    </a>
  </li>
  <li>
    <span>Click on `Download Build Tools` to download the executable</span>
  </li>
  <li>
    <span>
      {'Run the executable (proceed until you get the menu '}
      <i>Installing - Visual Studio Build Tools...</i>
      {')'}
    </span>
  </li>
  <li>
    <span>Check the first box for `Desktop development with C++`</span>
  </li>
  <li>
    <span>Click on `Install` (requires 6 Gb of disk space)</span>
  </li>
  <li>
    <span>
      Wait for the installation to complete. Once done, you can close the Visual
      Studio Build Tools window
    </span>
  </li>
</ol>

  ## <ChocolateyIcon /> Chocolatey
</Fragment>

<Fragment>
  We recommend using <ChocolateyIcon /> [Chocolatey](https://chocolatey.org/) as
  a package manager to install the necessary requirements for the application on{' '}
  <WindowsIcon /> Windows.
</Fragment>

<ol>

  <li>
    {'Type '}

      <PowershellIcon />
      {' PowerShell'}
    </i>

    {'Right-click on the '}

<Fragment>
  ## <PythonIcon /> Python
</Fragment>

<Fragment>
  For <PythonIcon /> Python installation, use the following command:
</Fragment>

```powershell
choco install python3 --version=3.10.11 -y
```

## üì¶ Other Dependencies

Install the additional dependencies required by the application:

  noPython
  noCCompiler
/>

Use the command below to install these dependencies:

```powershell
choco install nodejs-lts pnpm curl ffmpeg -y
```

